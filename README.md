# arithmetic-units

This project aims to deliver a step-by-step presentation of how the following arithmetic units work using a Java console application:<br/>

  Multipliers:
  * __Booth__<br/>
  * __Booth modified__<br/>
  * __Booth radix-4__<br/>

  Dividers:
  * __SRT Radix-2__<br/>
  * __SRT Radix-4__<br/>
